# Curso de Python

Bienvenido al repositorio del Curso de Python. Este repositorio contiene material educativo, ejemplos y ejercicios prácticos organizados por módulos y temas.

## Estructura del Curso

### Módulo 1

Este módulo cubre los fundamentos y herramientas esenciales, así como bibliotecas populares de Python.

- **Git**: Comandos básicos y flujo de trabajo.
- **Python**:
  - **Django**: Framework web de alto nivel.
  - **Docx2pdf**: Conversión de documentos Word a PDF.
  - **Docxtlp**: Generación de documentos Word a partir de plantillas.
  - **Gtts**: Texto a voz (Google Text-to-Speech).
  - **Jupyter**: Uso de Jupyter Notebooks.
  - **Matplotlib**: Creación de gráficos y visualizaciones.
  - **Os**: Interacción con el sistema operativo.
  - **POO**: Programación Orientada a Objetos (Clases, Herencia, Polimorfismo, etc.).
  - **PyGame**: Desarrollo de videojuegos.
  - **Smtplib**: Envío de correos electrónicos.

## Requisitos

- Python 3.x
- Las dependencias específicas de cada sub-módulo pueden requerir instalación (ej. `pip install django`, `pip install matplotlib`, etc.).

## Uso

Navega a través de las carpetas para encontrar scripts de ejemplo (`.py`) y cuadernos de Jupyter (`.ipynb`) con explicaciones y código ejecutable.

## Contribución

Si eres estudiante, recuerda trabajar en tu carpeta asignada y seguir las instrucciones de entrega.
